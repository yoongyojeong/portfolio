# Simple Portfolio Website

This is a simple responsive portfolio website made with HTML, CSS, and basic JavaScript.

## Pages

- `index.html`: Self-introduction page
- `index.html#ai-survey`: AI survey service form
- `activities.html`: Activity list page

## How to Use

1. Open `index.html` in a web browser.
2. Edit `Your Name`, the introduction, skills, and activities to match your own information.
3. Replace the Make.com webhook URL in `script.js` with your own Custom Webhook URL.
4. Keep `style.css` and `script.js` in the same folder as the HTML files.
